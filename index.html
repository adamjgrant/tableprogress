How to use Table Progress

This guide assumes you have table progress up and running as a Kickstrap app or have the CSS and JS resources hardwired into your site.

Here is the raw code for the above markup:

<table class="table table-bordered">
  <thead>
    <tr>
      <th>Column 1</th>
      <th>Column 2</th>
      <th>Column 3</th>
    </tr>
  </thead>
  <tbody>
    <tr class="progress">
      <td class="bar" colspan="3" style="width: 10%">
        10% Complete.
      </td>
    </tr>
    <tr>
      <td>Something</td>
      <td>Something</td>
      <td>Something</td>
    </tr>
    <tr class="progress">
      <td class="bar" colspan="3" style="width: 10%">
        10% Complete.
      </td>
    </tr>

    </tr>

</table>

Deconstruction

Table progress markup is designed to closely mirror the default progress bar markup from the Bootstrap docs.

Default bootstrap progress bars

<div class="progress">
  <div class="bar" width="50%"></div>
</div>

"Table progress" progress bars

<tr class="progress">
  <td class="bar" colspan="3" style="width: 50%">
    10% Complete.
  </td>
</tr>

Notice there are two key differences:

  1. tr and td are used instead of divs.
  2. A colspan is specified.
  3. Text is entered in the bar itself. This is an easy way to keep the sizing consistent.

You must also place the progress row above the row you want it to replace. And that's it!

Please note this currently only supports one-line rows.
